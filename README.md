# Using QRcodes to pre-load info in a Goolge Sheet
---
## download Node js

Click this to go to the Node website --> [Node JS](https://nodejs.org/en/download/)

After thats is finished downloading open Powershell and type:
  * npm init -y
  * mkdir src
  * cd src
  * nano app.js
  * npm install qrcode
